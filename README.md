# Software Development Processes Powered by AI Agents

This repository tracks progress through a course on modern software development processes, where each module is completed using a purpose-built AI agent to automate and enforce best practices.

# Chosen Kata

For Module 2, the repository will include software architecture documentation for the Birthday Greetings kata using the arc42 template and C4 diagrams.

**Birthday Greetings** is a kata about automatically sending birthday messages to people on the correct date based on stored personal data such as name, birthday, and email address.
